	Perso/VOTRENOM (sa Branch) : 
		--> C'est VOTRE Branch, vos modifications du projet ! 
		--> Cela permet de ne pas corrompre le projet global !
		--> Vous pouvez avoir des bugs, des modifications en cours, partir faire autre chose sans craintes !

	Main (LA Branch Global) :
		--> C'est LA Branch principal du projet !
		--> Vous ne devez y ajouter vos modifications que lorsque TOUT est stable dans votre Branch ET que vous avez l'accord du chef de projet !
			=> NE JAMAIS FAIRE L'ETAPE C SANS ACCORD OU EN AYANT DES BUGS !!!!
		--> Vous pouvez effectuez l'étape D sans craintes de corrompre la Branch Main !

	ETAPE A :: Avant de travailler sur VOS modifications :
		1. Double cliquer sur SA Branch

	ETAPE B :: Pour Commit les changements sur sa propre Branch :
		1. Double cliquer sur sa Branch
		2. Commit (Valider) :
			a. Tout indexer
			b. Ecrire un commentaire
			c. Valider
		3. Push (Envoyer) :
			a. S'assurer que Perso/VOTRENOM (sa Branch) est coché
			b. S'assurer que Perso/VOTRENOM (sa Branch) est sélec en face
			c. Envoyer

	ETAPE C :: Une fois que vos changements sont validés par la team :
		1. S'assurer que TOUT est stable et terminé sur votre branche
		2. S'assurer que TOUTES les modifications de votre Branch ont été soumis à l'étape B
		3. Double cliquer sur la Branch "main"
		4. Cliquer sur Fusionner :
			a. Sélectionner le dernier Commit de votre Branch (Perso/VOTRENOM)
			b. Cliquer sur "ok"
		

	ETAPE D :: Récupérer la Branch "main" sur votre Branch (Perso/VOTRENOM)
		1. Double cliquer sur Perso/VOTRENOM (sa Branch)
		2. Cliquer sur Fusionner :
			a. Sélectionner le dernier Commit de la Branch "main"
			b. Cliquer sur "ok"
		3. Vous pouvez à nouveau travailler sur votre Branch avec les modifications actuelles du projet (global)
